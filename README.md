# EventsAppBackend
A group-based event planning app built with Go, PostgreSQL, and React Native that allows friends to organize private events, manage RSVPs, and coordinate shared contributions like food, items, or money pools. The backend is designed around a lightweight REST API with JWT authentication, role-based authorization, and explicit event visibility.
